# Gitignore Templates

A collection of **`.gitignore` templates** for different technologies.

This repository provides simple and reusable templates to help you start projects without committing unnecessary files.

## Structure

Each template follows this format:

```
.gitignore-technology
```

Example:

```
LICENSE
README.md
.gitignore-angular
.gitignore-node
.gitignore-python
.gitignore-react
```

## Usage

Copy the template you need and paste it into your project's `.gitignore`.

Example:

```
cp .gitignore-node /your-project/.gitignore
```
