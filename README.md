# Gitignore Templates

A curated collection of **`.gitignore` templates for different technologies**.

This repository provides ready-to-use `.gitignore` files organized by **framework, language, and tool**, making it easier to start new projects with the correct files ignored.

---

## 📂 Repository Structure

Each template is stored as an individual file following this format:

```
.gitignore-technology
```

Example:

```
LICENSE
README.md

.gitignore-angular
.gitignore-node
.gitignore-python
.gitignore-react
.gitignore-docker
.gitignore-java
```

---

## 🚀 Purpose

The goal of this repository is to provide:

* Clean and reusable `.gitignore` templates
* Technology-specific ignore rules
* A simple reference for developers starting new projects

Instead of searching online every time, you can simply **copy the template you need**.

---

## 📥 How to Use

1. Find the template for your technology.
2. Copy the contents.
3. Paste it into the `.gitignore` file of your project.

Example:

```
cp .gitignore-node /your-project/.gitignore
```
